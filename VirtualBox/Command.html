Overview

Used by the rest of the virtualbox library to call shell commands. It also can be used to change the path for your VBoxManage program.

Changing VBoxManage Path

The rest of the library won't work without a proper path to VBoxManage, so it is crucial to set this properly right away. By default its set to VBoxManage which assumes that it is in your PATH.

VirtualBox::Command.vboxmanage = "/opt/local/bin/VBoxManage"

def shell_escape(str)
  str.to_s.gsub(/(?=[^a-zA-Z0-9_.\/\-\x7F-\xFF\n])/n, '\\').
           gsub(/\n/, "'\n'").
           sub(/^$/, "''")
end

def vboxmanage(command)
  result = execute("#{@@vboxmanage} #{command}")
  raise Exceptions::CommandFailedException.new(result) if !Command.success?
  result
end
